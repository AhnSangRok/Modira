## Modira 소개
MODIRA - 대구 사람들의 모임
## 프로젝트 개요
<br>
'모디라’는 대구 사투리로 ‘모여라’라는 뜻 입니다. 대구에서 거주하는 사람들에게
<br>
다양한 모임을 만들거나 참여할 수 있도록 도와주는 프로젝트입니다.

## 주요 기능
- 제공페이지 : 로그인, 회원가입, 메인페이지, POST 작성 페이지, POST 상세페이지
- JWT를 활용한 회원가입 & 로그인
- 게시물 CRUD 기능 

<br>

## 사용한 기술스택
# 프론트엔드
- React, Axios, Redux
# 백엔드
- Spring, JPA, MYSQL, GRADLE, GitHub, AWS, Spring Security
